# Zen Circuit

**Expo / React Native** meditation app: guided sessions, affirmations gallery, nature meditate flows, timers, and a calm gradient UI.

## Features

- Expo Router file-based navigation
- Tabs for meditate / affirmations
- Timer context for session duration
- NativeWind / Tailwind styling
- TypeScript throughout

## Tech

Expo · React Native · TypeScript · Expo Router · NativeWind

## Run

```bash
git clone https://github.com/d28035203/zen-circuit.git
cd zen-circuit
npm install
npx expo start
```
